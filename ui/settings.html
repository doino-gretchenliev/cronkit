Integration API

Control jobs programmatically. Authenticate every request with the key below, via Authorization: Bearer <key> or X-API-Key: <key>.

API key

{{.APIKey}}

Rotating immediately invalidates the old key.

Endpoints

Method & pathAction
GET /api/jobsList jobs with status
POST /api/jobs/{name}/runRun now
POST /api/jobs/{name}/cancelCancel the running execution
POST /api/jobs/{name}/disableDisable the job
POST /api/jobs/{name}/enableEnable the job

Example

curl -X POST -H "Authorization: Bearer {{.APIKey}}" \
  http://HOST/api/jobs/backup/run
